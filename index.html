⚙ GitHub Gist Sync

Setup in 3 Schritten:
1. Erstelle ein privates GitHub Gist auf gist.github.com
2. Generiere ein PAT unter GitHub → Settings → Developer settings → Personal access tokens mit Scope gist
3. Gib Gist-ID (aus der URL) und Token hier ein und klicke Speichern

Die Daten werden als skilltracker.json im Gist gespeichert. Das Token bleibt nur lokal im Browser — es wird nie in den Gist geschrieben.
